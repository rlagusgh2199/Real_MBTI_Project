Real MBTI

Real_MBTI는 설문이 아닌 실제 카카오톡 대화 기반 행동 데이터를 분석하여, 내 MBTI 성향과 대화 습관을 파악해주는 웹 서비스입니다.

  • 1 내 카카오톡 이름(닉네임) 입력
  • 2 카카오톡 대화 내보내기(.txt) 파일 업로드
  • 3 분석 시작 버튼 클릭 후 결과 확인

1. 분석 대상 설정

내 카카오톡 이름과 대화 내보내기 파일들을 업로드하면, 해당 사용자 기준으로 모든 분석이 진행됩니다.

대상

카카오톡 단톡/개인톡 내보내기(.txt) 파일

기준

“내 이름”과 일치하는 발화자 메시지들만 사용

결과

MBTI 유형 · 행동 패턴 · 신뢰도 · AI 리포트

단톡방에서 보이는 내 이름과 완전히 동일하게 입력해야 합니다.

카카오톡 > 대화 내보내기 > 텍스트(.txt)로 받은 파일들을 선택하세요.

📂
클릭해서 파일을 선택하거나,
여기로 드래그 앤 드롭 해주세요.

2. 분석 결과

한 줄 요약, MBTI 축별 분석, 행동 패턴, 데이터 메타 정보, LLM 리포트를 한 곳에서 확인할 수 있어요.
